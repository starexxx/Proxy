## Quick Start

### 1. Clone Repository

```bash
git clone https://github.com/starexxx/Proxy.git
cd Proxy
```
### 2. Start Proxy Bot
```
bash starexx.sh --proxy "start"
```
### 3. Stop Proxy Bot
```
bash starexx.sh --proxy "stop"
```
